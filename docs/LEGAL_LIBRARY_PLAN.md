# خطة المكتبة القانونية

تاريخ الاعتماد: 2026-07-20

هذه الخطة تضبط قسم `المكتبة القانونية` داخل تطبيق ميزان، باعتباره قسماً مرجعياً للبحث والحفظ والربط، وليس مركز تشغيل يومي.

---

## 1. القاعدة العامة

المكتبة القانونية قسم مستقل في التطبيق، هدفه حفظ وفهرسة المراجع القانونية السورية التي يحتاجها المحامي في عمله.

تدعم المكتبة:

```text
البحث
الفلترة
الفهرسة
فتح ملفات PDF / Word / نصوص
ربط مرجع بملف عند الحاجة
إضافة مرجع يدوي
```

---

## 2. الأقسام الرئيسية عند فتح المكتبة

عند فتح المكتبة القانونية تظهر بطاقات / أقسام:

```text
قوانين ومراسيم تشريعية
اجتهادات محكمة النقض
مجلة المحامون
أبحاث قانونية
تعاميم وقرارات إدارية
كتب ومراجع
```

يمكن لاحقاً إضافة أقسام أخرى إذا ظهرت حاجة عملية.

## 3. قوانين ومراسيم تشريعية

### 3.1 القاعدة

القوانين والمراسيم التشريعية تظهر ضمن تبويب واحد لأن المستخدم يبحث عملياً عن النص التشريعي، سواء كان:

```text

تعليمات تنفيذية
```

```text
فلتر حسب التصنيف القانوني
فلتر حسب السنة
```

## 5. مجلة المحامون

### 5.1 طريقة العرض

```text

```

### 5.2 بيانات كل عدد

```text
السنة

تاريخ الإصدار

```

### 5.3 مراحل الفهرسة

#### المرحلة الأولى

```text
إضافة العدد كملف PDF مع السنة ورقم العدد.
```

#### المرحلة الثانية

```text
فهرسة المقالات والاجتهادات داخل العدد.
```

فهرسة محتوى المجلة داخلياً تؤجل لأنها تحتاج قراءة أو OCR أو إدخال يدوي منظم.

## 7. تعاميم وقرارات إدارية

### 7.1 القاعدة الزمنية

تعتمد المكتبة القانونية إدخال التعاميم والقرارات الإدارية المهمة اعتباراً من:

```text
2024/12/08
```

مع إمكانية إدخال تعاميم أقدم إذا احتاجها المكتب.

### 7.2 طريقة العرض

```text
الأحدث أولاً

```

## 9. البحث العام داخل المكتبة

يجب أن تحتوي المكتبة على بحث عام:

```text
ابحث في المكتبة القانونية...
```

ويبحث في:

```text
القوانين والمراسيم
الاجتهادات
مجلة المحامون
الأبحاث
التعاميم والقرارات
الكتب والمراجع
```

وتظهر النتائج مصنفة حسب القسم.

---

## 10. ربط مرجع قانوني بملف

يمكن ربط مرجع قانوني بملف عند الحاجة.

أمثلة:

```text
ربط اجتهاد بدعوى
ربط قانون بمذكرة
ربط تعميم بإجراء / معاملة
ربط بحث قانوني بملف
```

الربط يكون بسيطاً:

```text
المرجع
الملف المرتبط
ملاحظات الربط
```

ولا يحتاج دورة عمل معقدة في المرحلة الأولى.


مع إمكانية إرفاق:

```text

صورة

```

## 12. مهمة جمع وتحميل المحتوى

المكتبة القانونية ليست مجرد شاشة؛ تحتاج مرحلة جمع محتوى وفهرسة قبل التنفيذ النهائي.

المهام الرئيسية:

```text
تنزيل القوانين والمراسيم التشريعية السورية.
تنزيل اجتهادات محكمة النقض والهيئة العامة.
تنزيل أعداد مجلة المحامون المتاحة.
جمع التعاميم والقرارات الإدارية من تاريخ 2024/12/08 وما بعد.
جمع أبحاث قانونية سورية مهمة.
تجهيز نماذج حازم الجزار ضمن مسار النماذج القانونية.
```

يجب أن تتم هذه المرحلة قبل إعداد خطة التنفيذ البرمجية النهائية، حتى تكون بنية المكتبة والنماذج متطابقة مع المحتوى الحقيقي.

---

## 13. ما لا نحتاجه في المرحلة الأولى

لا نحتاج في المرحلة الأولى إلى:

```text
تحميل تلقائي من الإنترنت داخل التطبيق.
تحديث تلقائي للمحتوى.
OCR إلزامي لكل PDF.
فهرسة داخلية كاملة لكل صفحة في كل مجلة.
محرر قانوني متقدم.
```

يمكن تنفيذ هذه الميزات لاحقاً.

---

## 14. الخلاصة التنفيذية

القواعد المعتمدة:

```text
المكتبة القانونية قسم مرجعي مستقل.
تظهر فيها: قوانين ومراسيم تشريعية، اجتهادات محكمة النقض، مجلة المحامون، أبحاث قانونية، تعاميم وقرارات إدارية، كتب ومراجع.
القوانين والمراسيم التشريعية في تبويب واحد.
اجتهادات محكمة النقض تقسم حسب الغرفة أو الهيئة.
مجلة المحامون تفهرس حسب السنة والعدد.
الأبحاث تفهرس حسب الباحث والموضوع والتاريخ.
التعاميم والقرارات الإدارية تعتمد من تاريخ 2024/12/08 افتراضياً.
كل مرجع يدعم ملفاً أو رابطاً أو نصاً.
يمكن ربط مرجع قانوني بملف.
مرحلة جمع وتحميل المحتوى تسبق خطة التنفيذ البرمجية النهائية.
```
