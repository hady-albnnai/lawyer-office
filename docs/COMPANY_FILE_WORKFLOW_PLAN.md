# خطة ملف الشركة وويزارد تأسيس الشركة

تاريخ الاعتماد: 2026-07-20

هذه الخطة تضبط ملف الشركة داخل التطبيق وفق واقع تأسيس الشركات في سوريا، مع الفصل بين:

```text
تأسيس الشركة
تعديلات ما بعد التأسيس
أرشيف الشركات القديمة
```

---

## 1. القاعدة العامة

ملف الشركة في التطبيق هو ملف مستقل برقم مكتب تلقائي:

```text
شركة/2026/0001
```

ويحفظ بيانات الشركة الأساسية ومؤسسيها وثبوتياتها وتعديلاتها اللاحقة، ويرتبط بالعقود والإجراءات والدعاوى والوكالات عند الحاجة.

---

## 2. بداية ويزارد الشركة

يبدأ تأسيس شركة جديدة من:

```text
مكتب العمل > إنشاء جديد > شركة
```

أما أرشيف الشركات القديمة فيبدأ من:

```text
ملفات المكتب > إدخال الأرشيف القديم
```

---

## 3. التصنيف التفصيلي للشركة

يجب إدخال تصنيف الشركة بشكل تفصيلي.

الأشكال الأساسية في سوريا:

```text
شركة تضامن
شركة توصية
شركة محاصة
شركة محدودة المسؤولية
شركة مساهمة مغفلة
```

مع إمكانية إضافة أو تفعيل تصنيفات أخرى عند الحاجة:

```text
شركة مدنية
شركة مناطق حرة
شركة قابضة
شركة خارجية
شركة مشتركة
تصنيف مخصص
```

القوائم قابلة للتوسع والتعطيل الآمن.

---

## 4. بيانات التأسيس الأساسية

الحقول المطلوبة في ويزارد التأسيس:

```text
اسم الشركة
تصنيف الشركة / شكلها القانوني
الشركاء المؤسسون
المفوض بالتأسيس
غاية الشركة
مقر الشركة
رأس المال إن وجد
الحصص / الأسهم إن وجدت
المدير / الإدارة إن وجدت
مدة الشركة إن وجدت
ملاحظات
```

### 4.1 المفوض بالتأسيس

في تأسيس شركة جديدة، المفوض بالتأسيس يكون غالباً:

```text
المكتب / المحامي / مستخدم من المكتب
```

مع إمكانية إدخال مفوض آخر عند الحاجة.

---

## 5. الشركاء المؤسسون

يدعم الويزارد عدة شركاء مؤسسين.

لكل شريك:

```text
شخص / جهة
الاسم
الصفة
الحصة / الأسهم إن وجدت
النسبة إن وجدت
رقم الهاتف
ملاحظات
```

---

## 6. الثبوتيات والمستندات

بعد إدخال بيانات الشركة، تأتي مهمة إدخال الثبوتيات.

القاعدة:

```text
كل بند ثبوتيات يترك للمستخدم مجالاً لإدخال اسم الوثيقة وزر لإرفاقها.
```

أي لا نفرض أسماء وثائق جامدة، بل نعطي مجموعات رئيسية مع إمكانية إضافة وثائق داخل كل مجموعة.

### 6.1 مجموعات الثبوتيات الأساسية

```text
طلب التأسيس
النظام الأساسي / عقد التأسيس
قرار الوزارة بالتأسيس
وثائق المالية
وثائق المقر
وثائق التأمينات الاجتماعية
وثائق غرفة التجارة
وثائق السجل التجاري
إيصالات متفرقة
موافقات خاصة
وثائق أخرى
```

### 6.2 عقد التأسيس والنظام الأساسي

حسب نوع الشركة:

```text
شركات الأموال: النظام الأساسي.
شركات الأشخاص: عقد التأسيس.
```

ويترك للمستخدم رفع الوثيقة المناسبة أو إدخال اسمها.

### 6.3 طريقة إدخال كل وثيقة

لكل وثيقة داخل أي مجموعة:

```text
اسم الوثيقة
تاريخ الوثيقة إن وجد
رقم الوثيقة إن وجد
الجهة الصادرة إن وجدت
إرفاق ملف
ملاحظات
```

## 8. تعديلات ما بعد التأسيس

التعديلات لا تحشر في ويزارد التأسيس.

تتم من ملف الشركة الجاري عبر زر:

```text
إضافة تعديل
```

### 8.1 فلسفة التعديل

لأن تعديلات الشركات كثيرة ومتنوعة، يعتمد التطبيق أداة تعديل مرنة:

```text
اختيار الحقل أو الموضوع الأساسي المراد تعديله.
إدخال اسم التعديل.
إرفاق الوثيقة أو المستندات.
إضافة ملاحظات.
```

## 9. الملفات المرتبطة بالشركة

ملف الشركة يمكن أن يرتبط بـ:

```text

إجراء سجل تجاري
إجراء تعديل شركة
دعوى تجارية
وكالة
إجراء مالي / ضريبي
```

الربط لا يعني دمج الملفات.

## 13. النواقص التحذيرية

أمثلة:

```text
لا يوجد تصنيف شركة
لا يوجد اسم شركة
لا يوجد شركاء مؤسسون
لا يوجد مفوض بالتأسيس
لا توجد غاية شركة
لا يوجد مقر
لا توجد وثيقة تأسيس / نظام أساسي
لا توجد وثائق سجل تجاري
أصل ورقي غير محدد
```

النواقص تحذيرية ولا تمنع الحفظ مبدئياً.

---

## 14. الخلاصة التنفيذية

القواعد المعتمدة:

```text
ملف الشركة يحفظ بيانات التأسيس الأساسية داخل التطبيق.
الثبوتيات تدار كمجموعات وثائق، وكل بند يسمح باسم وثيقة وإرفاقها.
تعديلات ما بعد التأسيس تتم من ملف الشركة الجاري بزر إضافة تعديل.
أداة التعديل مرنة: اختيار الحقل الأساسي، اسم التعديل، إرفاق الوثيقة.
الشركة ترتبط بعقود وإجراءات ودعاوى ووكالات عند الحاجة.
```
