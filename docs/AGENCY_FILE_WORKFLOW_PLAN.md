# خطة ملف الوكالة

تاريخ الاعتماد: 2026-07-20

هذه الخطة تضبط ملف الوكالة داخل التطبيق، وتشمل الوكالات القضائية والعدلية والخارجية والتفويضات، مع دعم العزل والاعتزال والربط بعدة ملفات.

---

## 1. القاعدة العامة

الوكالة كيان مستقل يمكن أن يرتبط بعدة ملفات:

```text
دعوى
إجراء / معاملة
عقد
شركة
ملف وكالة مستقل
```

رقم ملف الوكالة يولد تلقائياً:

```text
وكالة/2026/0001
```

---

## 2. نطاق ملف الوكالة

يشمل ملف الوكالة:

```text
الوكالات القضائية
الوكالات العدلية
الوكالات الخارجية
التفويضات
```

مع إمكانية إضافة تصنيفات مخصصة لاحقاً.

---

## 3. التصنيف الرئيسي للوكالة

```text
وكالة قضائية
وكالة عدلية
وكالة خارجية
تفويض / تكليف
تصنيف آخر
```

---

## 4. الوكالات القضائية

من الناحية العملية تعتمد التصنيفات التالية:

```text
سند توكيل عام
سند توكيل خاص
سند توكيل خاص شرعي
```

### 4.1 تصديقات / استعمالات الوكالة القضائية

كل وكالة قضائية قد تكون مصدقة / مستعملة وفق تصنيف عملي مثل:

```text
صلحية
بدائية
جنائية
تنفيذية
```

تسجل هذه كحقل مستقل أو قائمة متعددة القيم حسب الحاجة.

---

## 5. الوكالة العدلية

تسجل الوكالة العدلية كنوع مستقل لأنها غالباً تستخدم خارج القضاء، مثل:

```text
بيع
إدارة
رهن
تصرف عقاري
مراجعة دوائر
قبض
تمثيل أمام جهات
```

يجب دعم تحديد نطاق الوكالة نصاً أو من قائمة.

---

## 6. الوكالة الخارجية

الوكالة الخارجية تحتاج حقولاً خاصة لأنها تمر بإجراءات تصديق وتوطين.

### 6.1 حقول الوكالة الخارجية

```text
الدولة / مكان التنظيم
السفارة / القنصلية إن وجدت
تاريخ التنظيم
رقم الوكالة الخارجي إن وجد
تصديق وزارة الخارجية
تاريخ تصديق الخارجية
توطين لدى المحامي العام
تاريخ التوطين
رقم / مرجع التوطين إن وجد
هل الأصل مودع لدى المحامي العام؟
هل توجد نسخ مصدقة مستخرجة؟
ملاحظات
```

### 6.2 التوطين

المقصود بالتوطين:

```text
إيداع أصل الوكالة لدى المحامي العام، ثم يصبح بالإمكان استخراج نسخ مصدقة عنها.
```

يجب أن يظهر في حالة الوكالة الخارجية:

```text
بانتظار تصديق الخارجية
مصدقة من الخارجية
بانتظار التوطين لدى المحامي العام
موطنة لدى المحامي العام
نسخ مصدقة مستخرجة
```


---

## 9. العزل والاعتزال

يكفي في العزل أو الاعتزال إدخال:

```text
نوع الحدث: عزل / اعتزال
تاريخ العزل أو الاعتزال
السبب
وثيقة العزل أو الاعتزال
ملاحظات
```

### 9.1 أثر العزل أو الاعتزال

عند تسجيل عزل أو اعتزال:

```text
تحدث حالة الوكالة.
يظهر تنبيه على الملفات الجارية المرتبطة بها.
لا يمنع إدخال معلومات جديدة في الملف.
لا يوقف العمل تلقائياً.
```

النقص التحذيري:

```text
الوكالة المرتبطة بهذا الملف معزول عنها / تم الاعتزال منها.
```

---

## 10. إضافة وكالة داخل ملف آخر

عند إضافة وكالة داخل دعوى أو إجراء أو عقد أو شركة تظهر الخيارات:

```text
اختيار وكالة موجودة
إضافة وكالة جديدة
إضافة صورة وكالة دون بيانات كاملة
سأضيفها لاحقاً
```

الوكالة الموجودة تربط بالملف دون تكرار بياناتها.

---

## 11. الملفات المرتبطة

داخل ملف الوكالة يظهر تبويب:


أمثلة العلاقة:

```text

وكالة عقد

وكالة تأسيس
وكالة بيع

```


```text

```

```text

التصنيف الفرعي

فرع النقابة / الجهة
تاريخ التنظيم
الحالة
الملفات المرتبطة
النواقص
```

الأزرار:

```text
فتح الوكالة
ربط بملف
إضافة صورة / مستند
تسجيل عزل / اعتزال
إضافة أمر عمل
```

## 14. بطاقة الوكالة في الملفات المنتهية

تعرض:

```text
رقم ملف المكتب
وكالة
التصنيف الرئيسي
التصنيف الفرعي
الحالة
تاريخ الانتهاء / العزل / الاعتزال إن وجد
الملفات المرتبطة
هل توجد صورة سند؟

```

الأزرار:

```text
فتح الملف
كشف مستندات
محضر تسليم مستندات
إعادة فتح — للمالك / المدير فقط عند الحاجة
```

---

## 15. النواقص التحذيرية

أمثلة:

```text
لا توجد صورة سند الوكالة
رقم سجل الوكالة غير مدخل
الرقم الأبيض غير مدخل
فرع النقابة غير محدد
مندوب الوكالات غير محدد
الوكالة الخارجية غير مصدقة من الخارجية
الوكالة الخارجية غير موطنة لدى المحامي العام
الوكالة معزول عنها / تم الاعتزال منها
```

النواقص تحذيرية، ولا تمنع إدخال المعلومات، لكنها تظهر بوضوح.

---

## 16. الخلاصة التنفيذية

القواعد المعتمدة:

```text
ملف الوكالة يشمل القضائية والعدلية والخارجية والتفويضات.
الوكالات القضائية تصنف إلى سند توكيل عام، خاص، خاص شرعي.
تصديقات الوكالة القضائية تشمل صلحية، بدائية، جنائية، تنفيذية.
الوكالة الخارجية تدعم تصديق الخارجية والتوطين لدى المحامي العام وإيداع الأصل واستخراج نسخ مصدقة.
العزل والاعتزال يسجلان بتاريخ وسبب ووثيقة.
العزل والاعتزال يظهران كنقص تحذيري لا يمنع إدخال المعلومات.
الوكالة كيان مستقل قابل للربط بعدة ملفات.
```
