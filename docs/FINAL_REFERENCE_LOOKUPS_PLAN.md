# خطة القوائم المرجعية النهائية

تاريخ الاعتماد: 2026-07-20

هذه الخطة تضبط القوائم المرجعية النهائية اللازمة لتطبيق ميزان، بحيث تكون القوائم الافتراضية شاملة قدر الإمكان لواقع المحامي السوري، وتقلل حاجة المستخدم للإضافة اليدوية، مع إبقاء إمكانية الإضافة والتعديل والتعطيل وإضافة قوائم مخصصة.

---

## 1. القاعدة العامة

تدار القوائم المرجعية من:

```text
إدارة المكتب > القوائم المرجعية
```

مع إمكانية الإضافة السريعة من بعض الويزاردات عند الحاجة.

المكان الرسمي للإدارة:

```text
إدارة المكتب > القوائم المرجعية
```

أما الإضافة السريعة داخل الويزاردات فهي اختصار فقط.

---


### قائمة ذات حقول إضافية

تؤجل للمرحلة الثانية حتى لا يتوسع التنفيذ.

## 109. الحقول المخصصة

تؤجل الحقول المخصصة المرتبطة بالقوائم إلى مرحلة لاحقة.

المرحلة الحالية تعتمد:

```text
إضافة قائمة مخصصة
إدارة عناصرها
عدم ربطها تلقائياً بحقول مخصصة داخل كل شاشة
```

## 110. الخلاصة التنفيذية

القواعد المعتمدة:

```text
القوائم المرجعية مركزها إدارة المكتب.
تظهر الإضافة السريعة داخل الويزاردات عند الحاجة فقط.
كل قائمة تدعم الإضافة والتعديل والتعطيل والحذف الآمن.
كل عنصر يميز بين نظامي ومخصص.
لا حذف لأي عنصر مستخدم.
القوائم تقسم لمجموعات حتى لا تصبح الشاشة فوضوية.
تدعم إضافة قائمة مرجعية مخصصة.
الحقول المخصصة المرتبطة بالقوائم تؤجل لمرحلة لاحقة.
القوائم الافتراضية يجب أن تكون واسعة وشاملة قدر الإمكان لواقع المحامي السوري.
```
