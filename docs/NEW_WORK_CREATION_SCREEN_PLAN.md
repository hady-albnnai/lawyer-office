# خطة شاشة إنشاء جديد في مكتب العمل

تاريخ الاعتماد: 2026-07-20

هذه الخطة تضبط شاشة `إنشاء جديد` داخل مكتب العمل، باعتبارها البوابة الوحيدة للعمل الجديد بعد نزول التطبيق.

---

## 1. القاعدة الأساسية

زر / شاشة:

```text
إنشاء جديد
```

توجد فقط داخل:

```text
مكتب العمل
```

ولا توجد داخل:

```text
الملفات الجارية
الملفات المنتهية
التقويم
إدخال الأرشيف القديم
```

السبب:

```text
الملفات الجارية والمنتهية = عرض / بحث / تعامل.
إدخال الأرشيف القديم = ملفات موجودة قبل نزول التطبيق.
إنشاء جديد = عمل جديد بعد نزول التطبيق.
```

---

## 2. عنوان الشاشة

العنوان:

```text
إنشاء جديد
```

الوصف:

```text
ابدأ ملفاً أو عملاً جديداً بعد نزول التطبيق.
```

---

## 3. الخيارات الرئيسية

تظهر كبطاقات واضحة:

```text
دعوى
إجراء / معاملة
عقد
شركة
وكالة
أمر عمل عام
```

تظهر البطاقات حسب صلاحيات المستخدم.

---

## 4. بطاقة دعوى

العنوان:

```text
دعوى
```

الوصف:

```text
فتح ملف دعوى جديدة أو متابعة دعوى قائمة.
```

تفتح:

```text
ويزارد دعوى جديدة / دعوى متابعة
```

أول سؤال:

```text
دعوى جديدة
دعوى متابعة
```

---

## 5. بطاقة إجراء / معاملة

العنوان:

```text
إجراء / معاملة
```

الوصف:

```text
متابعة معاملة أمام جهة أو دائرة أو سجل أو نقابة.
```

تفتح:

```text
ويزارد إجراء / معاملة جديدة أو متابعة
```

أول سؤال:

```text
إجراء / معاملة جديدة
إجراء / معاملة متابعة
```

---

## 6. بطاقة عقد

العنوان:

```text
عقد
```

الوصف:

```text
إنشاء عقد أو مراجعة عقد جاهز أو متابعة عقد موقع.
```

تفتح:

```text
ويزارد العقد
```

أول سؤال:

```text
إنشاء عقد جديد
مراجعة / تعديل عقد جاهز
متابعة عقد موقع
```

---

## 7. بطاقة شركة

العنوان:

```text
شركة
```

الوصف:

```text
فتح ملف تأسيس شركة جديدة.
```

تفتح:

```text
ويزارد تأسيس شركة
```

في المرحلة الحالية، إنشاء الشركة من هنا يركز على:

```text
تأسيس شركة جديدة
```

أما تعديلات ما بعد التأسيس فتدار من ملف الشركة الجاري، وليس من شاشة إنشاء جديد.

---

## 8. بطاقة وكالة

العنوان:

```text
وكالة
```

الوصف:

```text
تنظيم أو متابعة وكالة قضائية أو عدلية أو خارجية.
```

تفتح:

```text
ويزارد وكالة جديدة
```

أول تصنيف:

```text
وكالة قضائية
وكالة عدلية
وكالة خارجية
تفويض / تكليف
```

---

## 9. بطاقة أمر عمل عام

العنوان:

```text
أمر عمل عام
```

الوصف:

```text
مهمة داخلية لا ترتبط بملف محدد.
```

أمثلة:

```text
مراجعة البنك
شراء قرطاسية
اجتماع داخلي
اتصال إداري
عمل تنظيمي
```

أمر العمل العام ليس ملفاً ولا يظهر ضمن ملفات المكتب، بل يظهر في مكتب العمل فقط.

---

## 10. ما لا يظهر في إنشاء جديد

لا تظهر الخيارات التالية داخل شاشة إنشاء جديد:

```text
إدخال أرشيف قديم
أرشيف جارٍ
أرشيف منتهٍ
شخص / جهة فقط
مستند فقط
```

السبب:

```text
الأرشيف القديم له تبويب مستقل.
الأشخاص والجهات يدخلون ضمن الملفات.
المستندات تدخل ضمن الملفات أو الأرشيف.
```

---

## 11. البحث قبل الإنشاء

يظهر في أعلى الشاشة حقل بحث:

```text
ابحث قبل إنشاء ملف جديد...
```

الغرض:

```text
تجنب التكرار
فحص وجود طرف سابق
فتح ملف موجود بدل إنشاء ملف جديد عند الحاجة
```

البحث يمكن أن يعرض:

```text
ملفات جارية مرتبطة بالاسم أو الطرف
ملفات منتهية
وكالات مرتبطة
عقود مرتبطة
إجراءات مرتبطة
شركات مرتبطة
```

هذا البحث تحذيري ومساعد، ولا يمنع الإنشاء تلقائياً.

---

## 12. رقم ملف المكتب

كل ويزارد يجب أن يعرض رسالة:

```text
سيتم توليد رقم ملف المكتب تلقائياً عند الحفظ.
```

أمثلة:

```text
دعوى/2026/0001
إجراء/2026/0001
عقد/2026/0001
شركة/2026/0001
وكالة/2026/0001
```

رقم ملف المكتب لا يدخله المستخدم ولا يعدل بعد الحفظ.

---

## 13. بعد الحفظ

بعد حفظ أي ملف جديد، تظهر خيارات مناسبة:

```text
فتح الملف
إضافة أمر عمل
إضافة مستند
العودة إلى مكتب العمل
```

حسب نوع الملف يمكن إضافة خيارات خاصة.

### مثال دعوى

```text
فتح ملف الدعوى
إضافة جلسة / موعد
إضافة أمر عمل
إضافة مستند
```

### مثال عقد

```text
فتح ملف العقد
فتح نسخة العقد
إضافة أمر عمل
إضافة مستند
```

---

## 14. الصلاحيات

تظهر بطاقات الإنشاء حسب صلاحيات المستخدم.

صلاحيات مقترحة:

```text
إنشاء دعوى
إنشاء إجراء / معاملة
إنشاء عقد
إنشاء شركة
إنشاء وكالة
إنشاء أمر عمل عام
```

---

## 15. علاقة إنشاء جديد بعمل اليوم والغد والتقويم

إذا أدخل المستخدم أثناء إنشاء الملف موعداً:

```text
اليوم
الغد
تاريخ لاحق
```

يجب أن يظهر العمل تلقائياً في:

```text
أعمال اليوم
عمل الغد
التقويم
```

بحسب التاريخ.

مثال:

```text
إنشاء إجراء جديد مع موعد مراجعة غداً
```

يدخل تلقائياً إلى:

```text
عمل الغد
```

---

## 16. عدم وجود إضافة سريع منفصلة

لا تعتمد شاشة `إضافة سريع` كمدخل مستقل في هذه المرحلة.

القاعدة:

```text
إنشاء الجديد يبدأ من مكتب العمل > إنشاء جديد فقط.
```

---

## 17. الخلاصة التنفيذية

القواعد المعتمدة:

```text
إنشاء جديد هو بوابة العمل الجديد فقط.
يوجد في مكتب العمل فقط.
لا يظهر في الملفات الجارية أو المنتهية أو التقويم أو الأرشيف.
يحتوي: دعوى، إجراء / معاملة، عقد، شركة، وكالة، أمر عمل عام.
لا يحتوي أرشيفاً ولا مستنداً ولا شخصاً مستقلاً.
يحتوي بحثاً قبل الإنشاء لفحص وجود ملفات أو أطراف سابقة.
كل ملف يحصل على رقم تلقائي عند الحفظ.
بعد الحفظ تظهر خيارات فتح الملف أو إضافة أمر عمل أو مستند.
البطاقات تظهر حسب صلاحيات المستخدم.
```
