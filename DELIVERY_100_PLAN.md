## 1) تعريف الاكتمال (Definition of Done للتسليم)

لا يُسلَّم التطبيق إلا إذا:

1. لا توجد شاشات Placeholder ظاهرة للمستخدم.
2. لا يوجد زر يعرض SnackBar نجاح دون حفظ/تنفيذ حقيقي.
3. لوحة اليوم تعرض بيانات حقيقية من SQLite (جلسات/مهام/نواقص/أوامر عمل).
4. «عمل جديد» يفتح مسارات إنشاء حقيقية (دعوى/عقد/شركة/إجراء/وكالة/أمر عمل).
5. أوامر العمل: إنشاء، طباعة PDF، نص واتساب، إدخال نتيجة، اعتماد — كلها تكتب في DB.
6. تسجيل نتيجة عمل/جلسة يحدّث الحالة ويمكن أن يولّد موعدًا قادمًا.
7. معالج أول تشغيل: مكتب فاضي + اسم المكتب/المحامي + كلمة مرور الزبون.
8. بناء Windows release + توثيق تشغيل صفحتين.
9. اختبارات آلية + Windows CI ناجحة.
10. التوثيق لا يدّعي اكتمالًا غير موجود.
