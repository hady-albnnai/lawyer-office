# دليل تشغيل الزبون — مكتب المحامي v1.0

## 1) التشغيل لأول مرة
1. شغّل `lawyer_office.exe`.
2. ستظهر شاشة **إعداد المكتب**.
3. أدخل: اسم المكتب، اسم المحامي، العنوان، الهاتف، كلمة المرور، سؤال الأمان.
4. **اترك بيانات تجريبية مغلقة** للعمل الحقيقي (مكتب فاضي).
5. احفظ وابدأ.

## 2) الشريط الجانبي (موحّد)
- اليوم
- الأجندة
- عمل جديد
- الملفات
- الأشخاص
- أوامر العمل
- المالية
- المستندات
- المكتبة القانونية
- البحث والتقارير
- الإعدادات

## 3) أوامر العمل
إنشاء → طباعة PDF → واتساب → إدخال نتيجة → اعتماد.  
كل خطوة تُحفظ محلياً.

## 4) النسخ الاحتياطي
الإعدادات → النسخ الاحتياطي → نسخ الآن. احفظ ملف ZIP خارج الجهاز.

## 5) ملاحظات
- التطبيق Offline بالكامل.
- لا يرسل بيانات إلى الإنترنت (إلا فتح رابط واتساب اختيارياً).
