# ملاحظات الإصدار v1.0 — مكتب المحامي

## للمستخدم
- شاشة إعداد أول تشغيل (مكتب فاضي افتراضياً).
- لوحة اليوم من البيانات الحقيقية.
- عمل جديد بمسارات إنشاء فعلية.
- أوامر عمل كاملة: إنشاء، PDF، واتساب، نتيجة، اعتماد (محفوظة محلياً).
- مالية، مكتبة قانونية، مستندات، دعاوى، أشخاص، بحث وتقارير.
- نسخ احتياطي من الإعدادات.

## تقنياً

- schemaVersion 3
- لا بذر تجريبي تلقائي إلا إذا اختاره المستخدم في أول تشغيل
